# Food Delivery Funnel & A/B Test Analysis

## Summary

A food delivery mobile application wanted to understand where users were abandoning the purchase journey and evaluate whether a new interface design improved conversion rates.

Using event-level user behavior data, I analyzed the customer journey, built a conversion funnel, validated an A/A/B experiment, and assessed the impact of a typography redesign through statistical hypothesis testing.

The analysis identified the primary friction point in the user journey and provided evidence-based recommendations for future product optimization and experimentation.

## Key Achievements

* Analyzed user behavior across multiple stages of a food delivery application.
* Built a complete conversion funnel from landing screen to successful payment.
* Identified the largest user drop-off point between the main screen and the offers screen.
* Measured conversion rates across the entire purchase journey.
* Validated experiment integrity through A/A testing.
* Evaluated the impact of a new typography design using statistical hypothesis testing.
* Determined that the redesign did not generate statistically significant changes in user behavior.
* Delivered data-driven recommendations for future product and growth initiatives.

---

## Key Findings

* The largest user drop-off occurred at the beginning of the user journey.
* Users who reached the offers screen demonstrated strong purchase intent.
* Approximately 47.7% of users completed the entire journey from landing page to successful payment.
* The A/A test confirmed that the experimental groups were properly randomized.
* The typography redesign did not significantly affect user behavior or conversion metrics.

---

## Business Recommendations

* Prioritize optimization of the initial user experience, where the highest abandonment occurs.
* Focus future experiments on features with greater potential impact on conversion.
* Continue using controlled experimentation to validate product decisions.
* Roll out the typography update if desired, as no negative impact on user behavior was detected.
